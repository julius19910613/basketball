# 业余篮球管理应用 - 功能设计文档

## 一、应用定位

面向业余篮球爱好者、球队队长、联赛组织者的综合管理工具，帮助管理球队、组织比赛、记录数据、促进交流。

---

## 二、核心功能模块

### 1. 用户系统

| 功能 | 说明 |
|------|------|
| 注册/登录 | 支持手机号、微信授权登录 |
| 个人资料 | 头像、昵称、身高、体重、位置（PG/SG/SF/PF/C）、擅长技能 |
| 球员卡 | 生成个人专属球员数据卡片 |

### 2. 球队管理

| 功能 | 说明 |
|------|------|
| 创建球队 | 队名、队徽、队服颜色、所属地区 |
| 球员管理 | 邀请入队、设置队长/副队长、球员号码分配 |
| 球队公告 | 发布通知、活动安排 |
| 球队相册 | 上传比赛照片、精彩瞬间 |
| 球队数据 | 胜负记录、场均得分、球队排名 |

### 3. 比赛管理

| 功能 | 说明 |
|------|------|
| 创建比赛 | 选择对手、时间、地点、比赛类型（友谊赛/联赛/锦标赛） |
| 报名签到 | 球员报名参赛、队长确认出场名单 |
| 比赛直播 | 文字直播、比分实时更新 |
| 数据记录 | 得分、篮板、助攻、抢断、盖帽、犯规、失误等 |
| 比赛回放 | 查看历史比赛详情和数据 |

### 4. 数据统计

#### 个人数据统计
- 场均得分、篮板、助攻、抢断、盖帽
- 投篮命中率（2分/3分/罚球）
- 效率值、正负值
- 近期状态趋势图

#### 球队数据统计
- 胜负记录、胜率
- 场均得分/失分
- 球队排名
- 球员贡献排行榜

### 5. 联赛/锦标赛系统

| 功能 | 说明 |
|------|------|
| 创建联赛 | 联赛名称、赛制（循环赛/淘汰赛）、参赛球队 |
| 赛程生成 | 自动生成赛程表 |
| 积分榜 | 实时更新排名 |
| 淘汰赛对阵图 | 可视化对阵图 |
| 奖项设置 | MVP、得分王、最佳防守等 |

### 6. 社交功能

| 功能 | 说明 |
|------|------|
| 发现球队 | 按地区搜索附近球队 |
| 约球功能 | 发布约球信息、接受挑战 |
| 动态广场 | 发布图文动态、点赞评论 |
| 消息通知 | 比赛提醒、入队申请、系统通知 |

### 7. 工具箱

| 功能 | 说明 |
|------|------|
| 计分器 | 比赛现场快速计分 |
| 计时器 | 比赛倒计时、24秒违例计时 |
| 抽签工具 | 随机分组、抽签对阵 |
| 战术板 | 简单战术绘制（进阶功能） |

---

## 三、用户角色与权限

| 角色 | 权限 |
|------|------|
| 普通球员 | 查看数据、报名比赛、发布动态 |
| 队长 | 球队管理、比赛创建、数据录入、球员审核 |
| 联赛组织者 | 创建联赛、管理赛程、审核球队 |
| 系统管理员 | 全平台管理 |

---

## 四、技术架构建议

### 前端
- **小程序端**：微信小程序（优先，便于分享和传播）
- **Web管理端**：Vue3 + TDesign（球队管理后台）

### 后端
- **云服务**：微信云开发 / CloudBase
- **数据库**：云数据库（球队、比赛、用户数据）
- **存储**：云存储（图片、视频）

### 第三方服务
- 地图服务：获取球场位置
- 短信服务：验证码通知
- 推送服务：比赛提醒

## 六、页面结构

```

├── 我的球队（快捷入口）
├── 近期比赛
├── 数据概览
├── 快速开始比赛

球队
├── 我的球队列表
├── 球队详情
│   ├── 球员名单
│   ├── 比赛记录

│   └── 相册
├── 创建球队

比赛
├── 比赛列表
├── 比赛详情
│   ├── 实时比分
│   ├── 数据面板
│   └── 球员数据
├── 创建比赛
├── 数据录入

发现
├── 附近球队
├── 约球信息
├── 动态广场

├── 个人资料
├── 我的数据

```

## 七、数据模型概览

- **User**（用户）
- **Team**（球队）
- **TeamMember**（球队成员关系）
- **Match**（比赛）
- **MatchStats**（比赛统计数据）
- **League**（联赛）
- **Post**（动态）

---

## 八、特色亮点

1. **轻量化设计**：专注业余场景，操作简单
2. **实时数据**：比赛数据实时记录，即时统计
3. **社交属性**：发现附近球友，组建篮球圈子
4. **数据可视化**：直观展示个人和球队成长轨迹
5. **免费使用**：基础功能免费，高级功能可选付费
